# MARVIN Setup

Welcome to MARVIN! Follow these steps to get started.

## Quick Start

1. Open a terminal in this folder
2. Run: `claude`
3. Type `/start` to begin

## First-Time Setup

When you first run MARVIN, it will:
- Ask for your name, role, and goals
- Create your personal workspace
- Set up communication preferences

## Commands

| Command | Description |
|---------|-------------|
| `/start` | Start your day with a briefing |
| `/end` | End session and save progress |
| `/update` | Save progress mid-session |
| `/report` | Generate weekly summary |
| `/help` | See all commands |

## Integrations

MARVIN can connect to:
- **Jira/Confluence** - Issue tracking & docs
- **Microsoft 365** - Email, calendar, files
- **Google Workspace** - Gmail, calendar, drive
- **Slack** - Team messaging

Type `/help` after setup to see how to connect integrations.
